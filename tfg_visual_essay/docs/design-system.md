# Sistema visual recomanat

## To visual

No copiïs The Pudding literalment. Agafa'n la lògica:

- jerarquia tipogràfica molt marcada
- ús fort de l'espai en blanc
- visuals integrats al relat
- colors amb caràcter però controlats
- ritme de lectura clar

## Traducció al teu TFG

### Paleta orientativa

- fons crema o os trencat
- negre suau per al text
- rosa intens o magenta per a l'afecte, el fandom i el ship
- groc càlid o taronja per moments de viralitat o intensificació
- violeta o blau per al marc teòric i els dispositius de plataforma

### Tipografia

- una sans serif contemporània per al cos
- una display potent per a grans titulars, o bé una sans molt pesada

### Estil de visuals

- diagrames nets
- captures tractades amb coherència de color
- SVG abans que PNG quan sigui possible
- combinar gràfics amb anotacions textuals curtes

## Evita

- massa text seguit
- efectes gratuïts
- més de dues idees fortes dins el mateix tram de scroll
- simular exactament la identitat de The Pudding
