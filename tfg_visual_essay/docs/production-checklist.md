# Checklist de producció

## Abans de dissenyar

- [ ] Tenir l'argument global resumit en 5 o 6 frases
- [ ] Decidir els capítols definitius
- [ ] Seleccionar només les cites imprescindibles
- [ ] Decidir quins visuals faràs de debò i quins no

## Abans de programar visuals finals

- [ ] Tenir netejades les dades o materials qualitatius
- [ ] Fer esbossos a mà o a Figma de cada visual
- [ ] Escriure peus de visual molt breus

## Abans de publicar

- [ ] Revisió ortogràfica completa
- [ ] Mobile first mínim acceptable
- [ ] Contrast suficient
- [ ] Temps de càrrega raonable
- [ ] Metadades i títol correctes
- [ ] Crèdits i bibliografia web condensada

## Publicació acadèmica

Et recomano tenir dues versions:

1. **Versió completa interactiva**, per presentar i ensenyar
2. **Versió textual/resumida**, per poder annexar al TFG o documentar-ne l'estructura
