# Model de continguts

## Per cada capítol, prepara això

### 1. Idea central

Una sola frase que expressi què demostres.

### 2. Text curt

Entre 80 i 180 paraules. En web llarga, menys és millor.

### 3. Evidència

- una cita
- una dada
- un cas concret
- una captura o reconstrucció

### 4. Visual associat

Defineix quin format necessita:

- timeline
- xarxa
- matriu
- cards
- stepper scrolly
- collage d'interfícies

### 5. Funció argumentativa del visual

El visual ha de:

- explicar una relació
- mostrar una distribució
- donar context temporal
- comparar espais o pràctiques
- intensificar emocionalment un punt

## Exemple de fitxa

```yaml
id: shipping
idea: El ship funciona com a pràctica de producció afectiva i narrativa.
text: El shipping no és només una fantasia privada, sinó una manera de llegir, prolongar i reorganitzar relacions que el programa deixa obertes o disponibles per a la interpretació.
quote: "..."
visual: network
function: mostrar connexions entre fans, relats i figures del programa
```
