# Presentació Visual de la Tesi

Aquesta és una pàgina web estàtica per presentar la teva tesi en format visual, inspirada en l'estil de data journalism de The Pudding (https://pudding.cool/).

## Estructura del Projecte

- `main.html`: Pàgina principal amb l'estructura de la tesi.
- `style.css`: Estils CSS per a un disseny net i modern.
- `script.js`: Scripts JavaScript amb D3.js per a visualitzacions interactives.

## Com Utilitzar

1. Obre `main.html` en un navegador web per veure la presentació.
2. Edita el contingut de `main.html` per afegir el text de la teva tesi.
3. Modifica les dades en `script.js` per actualitzar les visualitzacions amb les teves dades reals.
4. Personalitza els estils en `style.css` si cal.

## Dependències

- D3.js (v7): Inclòs via CDN en `main.html`.

## Troubleshooting

- Si les visualitzacions no es carreguen, assegura't que tens connexió a internet per carregar D3.js.
- Per a navegadors antics, considera utilitzar una versió més antiga de D3.js.
- Si vols afegir més visualitzacions, consulta la documentació de D3.js: https://d3js.org/

## Inspiració

Basat en l'estil visual de The Pudding, que utilitza històries visuals per explicar idees complexes amb dades.
