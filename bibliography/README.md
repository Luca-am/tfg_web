# Bibliography Conventions

Aquest directori conté PDFs font i fitxes HTML associades.

## Estructura

- `books/`: fitxes HTML individuals
- subcarpetes temàtiques (`FANDOM-RELIGIÓ`, `MARC TEÒRIC GENERAL`, etc.): PDFs originals

## Convencio de fitxes HTML (`books/`)

Format recomanat:

- `cognom-any-titol-curt.html`
- Exemple: `caceres-2002-operacion-triunfo.html`

## Convencio de metadades a la fitxa

- `Autor: Nom Cognom (ANY)`
- `Data: ANY`
- Si no hi ha any confirmat:
  - `Autor: Nom Cognom (s.d.)`
  - `Data: s.d.`

## Recomanacions per PDFs nous

- Evita canviar el nom dels PDFs ja enllaçats sense actualitzar `bibliography.html` i la fitxa de `books/`.
- Si afegeixes un PDF nou, mantén una versió de nom estable i després crea la fitxa HTML amb slug.
