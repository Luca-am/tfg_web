Editor de codi utilitzat per construir i mantenir aquest lloc web del TFG.

Privacitat:
7.5/10

Programari de codi obert amb telemetria opcional desactivable. El codi roman en local.

Nivell de privacitat

Visual Studio Code és un editor de codi obert desenvolupat per Microsoft. Per defecte recull telemetria anònima d'ús, però aquesta pot ser desactivada completament des de la configuració. Tot el codi i els fitxers del projecte es mantenen en local, sense cap sincronització obligatòria.

Com descarregar-lo

Descarrega VS Code des de https://code.visualstudio.com/. Disponible per a Windows, macOS i Linux. És gratuït i de codi obert.

Com l'he utilitzat i per què m'ha servit

He emprat Visual Studio Code per escriure i editar tot el codi HTML, CSS i JavaScript d'aquest lloc web. La integració amb Git, les extensions de previsualització i el resaltat de sintaxi han facilitat enormement el procés de desenvolupament i manteniment del projecte.
