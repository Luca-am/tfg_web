# Interviews Media Structure

Aquest directori centralitza tots els materials d'entrevistes.

## Estructura

- `profiles/`: fotos o avatars de perfil
- `audio/`: audios finals publicables
- `transcripts/`: transcripcions finals (PDF o TXT)
- `pending/audio/`: audios pendents de revisar/netejar
- `pending/transcripts/`: transcripcions pendents de revisar

## Convencio de noms

Per fitxers finals, usa:

- Audio: `YYYY-MM-DD-nom-persona-plataforma-metode.ext`
- Transcripcio: `YYYY-MM-DD-nom-persona-transcript.ext`
- Perfil: `nom-persona.ext`

Exemple:

- `2026-04-08-reme-ii-google-meets-call.mp3`
- `2026-04-08-reme-ii-transcript.pdf`
- `reme-ii.jpg`

## Notes

- Evita espais i accents en noms de fitxer.
- Mantingues minuscules i guions.
- Si falta material, desa'l primer a `pending/` i mou-lo a carpeta final quan estigui validat.
